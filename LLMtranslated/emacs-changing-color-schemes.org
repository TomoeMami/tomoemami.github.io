[2025-11-28 Fri]: Since ~ef-themes~ is now based on ~modus-themes~, there have been many changes.
[2025-06-20 Fri]: Switched to using the =seq-random-elt= function to randomly pick values from a list.

These past few days, I switched my Emacs color scheme from [[https://github.com/greduan/emacs-theme-gruvbox][gruvbox-light-soft]], which I had used for nearly 10 years, to the randomly switching [[https://github.com/protesilaos/ef-themes][ef-themes]].

* Randomly Switching Themes
I selected a few =ef-themes= themes that I find visually appealing, referred to the official configuration to randomly switch themes when starting Emacs, and placed the configuration in ~init.el~.

#+begin_src elisp
(use-package modus-themes)
(use-package ef-themes
  :init
  ;; Use only ef-themes
  (ef-themes-take-over-modus-themes-mode 1)
  :config
  ;; Remove bold from org headings
  (setq modus-themes-headings ; read the manual's entry or the doc string
      '((0 . (regular 1))
        (1 . (regular 1))
        (2 . (regular 1))
        (3 . (regular 1))
        (4 . (regular 1))
        (5 . (regular 1)) ; absence of weight means `bold'
        (6 . (regular 1))
        (7 . (regular 1))
        (t . (regular 1))))
  ;; Disable bold and italic
  (setq modus-themes-mixed-fonts t
        modus-themes-variable-pitch-ui nil
        modus-themes-bold-constructs nil
        modus-themes-italic-constructs nil)
  ;; ;; Disable all other themes to avoid awkward blending:
  ;; (mapc #'disable-theme custom-enabled-themes)
  ;; Random load the theme of choice:
  (modus-themes-load-theme (seq-random-elt '(ef-arbutus ef-cyprus ef-day ef-eagle ef-elea-light ef-frost ef-kassio ef-light ef-maris-light ef-melissa-light ef-reverie ef-spring ef-summer ))))
#+end_src

* Configuring Automatic Color Retrieval
Use the =modus-themes-get-color-value= macro to obtain raw color strings. Color names can be viewed with =ef-themes-list-colors= or =ef-themes-list-colors-current=.

** nano-modeline Color Scheme
#+begin_src elisp
(use-package nano-modeline
  :demand t
  :custom-face
  ;; Customize nano-modeline colors
  (nano-modeline-status ((t (:background ,(modus-themes-get-color-value 'fg-alt) :foreground ,(modus-themes-get-color-value 'bg-main)))))
  (nano-modeline-inactive ((t (:background ,(modus-themes-get-color-value 'bg-inactive) :foreground ,(modus-themes-get-color-value 'fg-dim)))))
  (nano-modeline-active ((t (:background ,(modus-themes-get-color-value 'bg-alt) :foreground ,(modus-themes-get-color-value 'fg-main))))))
#+end_src

** sis Cursor Color
The initial English state uses dark gray, while the Chinese state uses a special color, consistent with the nano-modeline color scheme.

#+begin_src elisp
(use-package sis
  :demand t
  :config
  (setq sis-other-cursor-color (modus-themes-get-color-value 'fg-alt))
  (setq sis-default-cursor-color (modus-themes-get-color-value 'border)))
#+end_src
