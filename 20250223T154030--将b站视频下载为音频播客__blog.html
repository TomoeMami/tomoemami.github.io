将b站视频下载为音频播客

[2025-02-23 日 15:40]

有的B站视频，文案精心设计，但是画面就是一个又一个的表情包，完全可以当成播客来听。但是B站官方手机APP的听视频功能很奇怪，体验远不如现有的播客客户端。所以我们把B站的视频下载下来，用播客客户端听。

[2026-01-06 二 17:25]：目前已经改用直接在B站创建一个收藏夹，然后通过底栏右侧「我的」—「听视频」入口直接播放收藏夹视频，还能和网页版同步进度。

[2026-02-04 三 19:40]： 目前回到了自建audiobookshelf收听播客与音频的方式，「听视频」功能有一个重大问题，在自动播放的时候会覆盖掉下一个视频的播放进度

1. 下载B站视频

安装Bilibili-Evolved脚本，利用其中的下载功能下载音频。具体方式请查看脚本说明。

2. 搭建播客软件

这里选用的是Folder2Podcast，直接按照官方说明clone到本地即可。

然后修改 package.json 里的 scripts 后面选项，把 dev 的内容按照你需要的配置修改并保存。

比如 AUDIO_DIR ，就是指向音频文件夹，软件会在这个路径里找子文件夹，检查其中的音频。

比如 BASE_URL ，要改为服务运行机器的局域网或公网IP

再输入 npm run dev 就能启动了。

3. 创建子文件夹的说明

这个软件的功能，是在给定的 AUDIO_DIR 下寻找子文件夹，将每个子文件夹作为一个单独的RSS订阅源。

我们以 AUDIO_DIR=~/bilipod/ 为例，新建文件夹 ~/bilipod/bilibili ，在其中新建 podcast.json ：

{
    "title": "B站音频",
    "description": "B站视频转为的音频",
    "author": "Me",
    "email": "author@example.com",
    "language": "zh-cn",
    "category": "Technology",
    "explicit": false,
    "websiteUrl": "https://www.bilibili.com",
    "titleFormat": "full",
    "useMTime": true
  }

这样，软件就会自动识别标题等内容了。

然后进入 BASE_URL ，就可以看见一个友善的网页界面，选择复制即可。
