+blog+一加手机解锁刷入KernelSU

[2024-08-17 周六 20:00]

1. 准备工作

下载以下资源：

对应手机系统当前版本的官方ROM，用于提取 init_boot.img

Platform tools ，用来操作fastboot

2. 开始解锁BL

解锁和上锁BL都会格式化手机，提前做好数据备份

  1. 开启「开发者模式」，打开「OEM解锁」
  2. 用 原装数据线 连接电脑，使用adb工具 adb devices ，确认有输出
  3. 执行 adb reboot bootloader ，进入bl界面
  4. 执行 fastboot flashing unlock ，用音量键移动，电源键确认解锁

3. 刷入img文件

  1. 下载对应ROM，用Payload-Dumper 执行 python payload_dumper.py --images "init_boot" payload.bin 解压出ROM文件
  2. 找到 init_boot.img 并将其传输到手机上
  3. 手机下载安装并打开 KernelSU管理器，选中输入的 init_boot.img ，按流程修补文件
  4. 将输出的 KernelSU***.img 拷贝到 adb.exe 所在路径并重命名为 init_boot.img
  5. 执行 adb reboot bootloader ，进入bl界面
  6. 执行 fastboot flash init_boot init_boot.img ，将修补后的文件刷入手机
  7. 执行 fastboot reboot ，重启设备
