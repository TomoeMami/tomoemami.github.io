+blog+在手机上使用rime输入法及语言模型

[2024-12-18 周三 12:51]

整体操作很简单，主要结合以下两个帖子，这里简单记录

尊重隐私的输入方案，fcitx5-android + 雾凇拼音 快速上手

小企鹅输入法安卓版主题 Fcitx5-Andoird theme

1. 安装

在 GitHub releases 页面下载对应架构的应用本体（app）和插件（plugin.rime），并安装。

或者在F-droid上一键安装本体和rime插件

2. 配置

在 app > 输入法中，右下角可以添加“中州韵”输入法。添加后在“中州韵”右侧的设置页面中，可以打开其数据目录。

删除 数据目录/data/rime 下所有文件，然后把桌面rime的用户文件夹中所有内容，除 build 文件夹和 *.userdb 文件夹外，复制到 数据目录/data/rime 中。

随便找一个输入框唤出 fcitx5-android 输入法，点击左上角箭头展开菜单，点击右边三个点打开更多菜单，点击“重载配置”；或者杀死其后台，并重新打开启动器上的输入法图标。输入法会检测到配置文件更新，并提示正在重新部署。

完成后，底部会弹出 toast，提示部署完成，即可使用。

3. 安装主题

将主题压缩包下载至手机——直接下载 zip 包即可，截图和 json 文件仅供预览和修改用，实际上 zip 包里面就是同名 json 文件。

手机键盘弹出→工具栏展开→三个省略号→主题，或者小企鹅APP→主题

选择新增主题→导入文件，选择主题压缩包。

4. 语言模型

按照先前教程为rime输入方案添加语言模型支持 配置好之后无需额外操作，复制进手机的输入法配置文件夹即可。
